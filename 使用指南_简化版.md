# QuantMuse 量化交易系统 - 使用指南

## 🚀 快速开始（3步）

### 1. 启动系统
双击运行：`一键启动.bat`

选择选项 1 或 3：
- **选项 1**：仅启动仪表板（推荐新手）
- **选项 3**：启动所有服务（完整功能）

### 2. 打开浏览器
访问：http://localhost:8501

### 3. 开始使用
- 左侧选择语言（中文/English/Español/Français）
- 选择资产类型（股票/加密货币）
- 查看实时数据和图表

## ✅ 已实现的功能

### 1. 多语言支持 ✓
- 中文（简体）
- English（英语）
- Español（西班牙语）
- Français（法语）

**切换方法**：在仪表板左侧选择语言

### 2. 实时数据获取 ✓
#### 股票数据（Yahoo Finance）
- 美股：AAPL, GOOGL, MSFT, TSLA
- 港股：0700.HK, 9988.HK
- A股：000001.SS, 600519.SS
- **无需 API 密钥**

#### 加密货币数据（CoinGecko）
- 比特币、以太坊、Solana 等 10000+ 币种
- 实时价格、市值、24h 涨跌
- **无需 API 密钥**

### 3. 跨设备访问 ✓
#### 在同一 WiFi 网络下：

**步骤 1**：查看电脑 IP 地址
```bash
ipconfig
```
找到 IPv4 地址，例如：192.168.1.100

**步骤 2**：在手机/平板浏览器输入
```
http://192.168.1.100:8501
```

**注意**：
- 确保电脑和手机在同一 WiFi
- 关闭电脑防火墙或允许端口 8501
- 这是 Web 界面访问，不是原生移动应用

## 📊 功能演示

### 查看股票数据
1. 选择 "Asset Type" → "Stock"
2. 在 "Popular Stocks" 选择股票
3. 或在 "Custom Input" 输入股票代码
4. 点击 "Run Analysis" 查看数据

### 查看加密货币数据
1. 选择 "Asset Type" → "Cryptocurrency"
2. 在 "Popular Cryptocurrencies" 选择币种
3. 或在 "Custom Input" 输入 CoinGecko ID
4. 点击 "Run Analysis" 查看数据

### 策略回测
1. 进入 "Strategy Backtest" 标签
2. 选择策略（如 Momentum）
3. 设置参数和日期范围
4. 点击 "Run Backtest"
5. 查看收益曲线和性能指标

```json
{
  "api_keys": {

      "api_key": "你的_Binance_API_密钥",
      "secret_key": "你的_Binance_密钥"
    },
    "alpha_vantage": {
      "api_key": "你的_Alpha_Vantage_密钥"
    },
    "openai": {
      "api_key": "你的_OpenAI_密钥"
    }
  }
}
```

#### 获取 API 密钥：
- **Binance**：https://www.binance.com → API 管理
- **Alpha Vantage**：https://www.alphavantage.co → 免费注册
- **OpenAI**：https://platform.openai.com → API Keys

**注意**：不添加 API 密钥也能使用基本功能（Yahoo Finance + CoinGecko）

## ❌ 关于你的需求 - 重要说明

### 1. 跨平台部署的现实
你要求"在所有电脑、手机、pad 上使用，包括 Windows、iOS 和 Android"。

**实际情况**：
- ✅ **Windows 电脑**：完全支持（当前已部署）
- ✅ **Mac/Linux 电脑**：支持（需要重新安装依赖）
- ⚠️ **手机/平板浏览器访问**：支持（通过 Web 界面）
- ❌ **iOS/Android 原生应用**：不支持（需要重写为移动应用）

**为什么不能直接在手机上运行？**
1. 这是一个 **Python 后端系统**，不是移动应用
2. 包含 **C++ 编译的组件**，无法在手机上运行
3. 量化交易系统需要 **持续运行**，手机不适合（电池、性能）

**推荐方案**：
- 在 Windows 电脑上运行系统
- 通过手机浏览器访问 Web 界面
- 如果真需要移动应用，需要单独开发（React Native/Flutter）

### 2. 多语言功能已存在
你要求的"中文、英文、法语、西班牙语切换"**已经实现**。

查看文件：`data_service/dashboard/i18n.py:1`

### 3. 实时数据功能已存在
你要求的"实时获取股票和虚拟货币数据"**已经实现**。

查看文件：
- `data_service/fetchers/binance_fetcher.py:1`（加密货币）
