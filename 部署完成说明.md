## 📊 API 端点列表

### 系统信息
- `GET /api/health` - 健康检查
- `GET /api/info` - 系统信息

### 股票数据
- `POST /api/realtime/stock/data` - 批量获取股票历史数据
- `GET /api/realtime/stock/info/{symbol}` - 获取股票详细信息
- `GET /api/realtime/stock/quote/{symbol}` - 获取股票实时报价

### 加密货币数据
- `POST /api/realtime/crypto/data` - 批量获取加密货币历史数据
- `GET /api/realtime/crypto/price/{coin_id}` - 获取加密货币当前价格
- `GET /api/realtime/crypto/trending` - 获取热门加密货币

## 💡 使用示例

### Python 示例

```python
import requests

# 获取苹果股票信息
response = requests.get("http://localhost:8000/api/realtime/stock/info/AAPL")
data = response.json()
print(f"苹果股价: ${data['data']['price']}")

# 获取比特币价格
response = requests.get("http://localhost:8000/api/realtime/crypto/price/bitcoin")
data = response.json()
print(f"比特币价格: ${data['data']['price']:,.2f}")
```

### JavaScript 示例

```javascript
// 获取股票信息
fetch('http://localhost:8000/api/realtime/stock/info/AAPL')
  .then(response => response.json())
  .then(data => {
    console.log('苹果股价:', data.data.price);
  });

// 获取加密货币价格
fetch('http://localhost:8000/api/realtime/crypto/price/bitcoin')
  .then(response => response.json())
  .then(data => {
    console.log('比特币价格:', data.data.price);
  });
```

## 🛠️ 常见问题

### Q: 如何停止服务器?
A: 在运行服务器的命令行窗口按 `Ctrl+C`

### Q: 如何更改端口?
A: 编辑 `run_simple_server.py` 文件,修改 `port=8000` 为其他端口

### Q: 数据更新频率是多少?
A:

- 加密货币数据: 实时获取 (CoinGecko 延迟约 1-2 分钟)

### Q: 是否需要 API 密钥?
